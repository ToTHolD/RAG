# RAG
for fedya
Class TextCLeaner - Очистка текста промпта юзера от шумов и разбитие на чанки текста из бд
Здесь функции:

def clean_text - с помощью регулярных выражений убираем слова и буквы и символы которые мешают llm модели

def chunk_text - разбиваем текст из бд на чанки с помощью библиотеки langchain(здесь нужно доделать чтобы способы разбиения для узкой и широкой базы были разные)

class EmbeddingVectorizer - преобразуем промпт юзера и бд в циферки(вектора)

def __init__ - просто функция сборщик инициализирует некоторые переменные

def encode - преобразует текст в вектора с помощью модели 'paraphrase-multilingual-MiniLM-L12-v2' и нормализацией L2

def encode_single - преобразуем запрос юзера в вектор 

Class RAGDatabase - класс для работы с дб

Def __init__ - сборщик

def add_documents - собирает бд в единый массив векторезует их и разбивает на чанки

Def _process_documents - приватная функция непосредственно связан с прошлой функцией здесь вызываем функции очистки и разбития на чанки для бд и функцию для векторизации

def search - здесь мы сравниваем векторизованный запрос юзера с векторами из бд по косиноснуму сходтству отбираем самые подходящие чанки контекста, сначала ищем в первой, если сходство меньше порога обращаемся ко второй

def get_chunk_count - кол-во чанков

def create_rag_prompt - делаем системный промпт для модели llama что бы она действовала по заданной инструкции 

Class OllamaRAG - непосредственно RAG система

def __init__ - функция сборщик

def _initialize_database - приватная функция инициализирует дб(лучше считывать их из файла)
и вызывает функцию для сборки бд

def _get_query_emdedding - приватная функция векторизация запроса юзера с нормализацией

Def _query_ollama - отправляем запрос к платформе ollama с заданными параметрами

def query - основная функция RAG системы здесь все функции собираются воедино и выполняют поставленную задачу от RAG системы то есть Запрос юзера векторизуем, сравниваем с векторами из бд, находим лучший подходящий контекст из бд, отправляем запрос в llama 3.2 с указанным контекстом.
